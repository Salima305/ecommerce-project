# 🛒 E-Commerce Web Application

A full-stack e-commerce platform for fashion jewellery built using Node.js, Express.js, MongoDB, and EJS.

## 🌟 Features

### User Side
- User authentication and profile management
- Product listing and detail pages
- Shopping cart with dynamic quantity updates
- Coupon system with discount calculation
- Secure checkout flow
- Cash on Delivery and Razorpay payment integration
- Order tracking, cancellation and return requests

### Admin Panel
- Product and order management
- Order status updates
- Analytics dashboard with sales charts and top-selling products

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Bootstrap, EJS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Payments:** Razorpay

## 🚀 How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with your credentials
4. Run `node app.js`
